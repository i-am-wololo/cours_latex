\chapter{Mémoire}

La mémoire RAM est là où le système et ses processus sont chargés durant le fonctionnement d'un ordinateur.

De nos jours, un processeur est capable d'adresser la mémoire de deux façons différentes:
\begin{itemize}
	\item Le mode réel
	\item le mode virtuel (aussi appelé mode protégé)
\end{itemize}
